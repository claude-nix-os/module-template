# Changelog

All notable changes to this module will be documented in this file.

## [1.0.0] - 2025-01-01

### Added
- Initial module template with ClaudeOS module contract types
- Example panel component demonstrating UI integration
- Example API route handler following Next.js patterns
- Module validation test suite
- CI workflow for test, typecheck, and build on every push
- Publish workflow for auto-publishing to GitHub Package Registry on tag push
- Nix flake for reproducible development environment
